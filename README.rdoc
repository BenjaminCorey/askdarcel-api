== README

{<img src="https://travis-ci.org/ShelterTech/sfhomeless-api.svg" alt="Build Status" />}[https://travis-ci.org/ShelterTech/sfhomeless-api]

=== Prerequisites

* PostgreSQL
* Ruby 2.2.2
* Bundler

=== Setup

  ./bin/setup

=== Running

  ./bin/rails server

See

  ./bin/rake routes

for available routes.

=== Testing

  ./bin/rake

=== Deployment

  heroku login
  heroku git:remote -a sfhomeless-api
  heroku push
